github创建仓库并使用git上传本地代码

创建本地仓库和github仓库

在Github上创建仓库并关联本地仓库，创建一个名为hello的github仓库

将标红处复制一下

输入命令完成关联

git remote add origin git@github.com:shenhuagg555/hello.git

添加一个README.md文件来描述内容，使用git add <filename>添加文件

这里报了一个错误，需要我们设置邮箱和用户名，根据提示设置即可

配置ssh

但是由于我们没有设置公钥，于是又出现了以下错误

输入ssh-keygen -t rsa生成公钥和私钥，连续按三次回车生成，公钥和私钥保存在~/.ssh 里，再输入cat ~/.ssh/id_rsa.pub查看生成的公钥，将其复制，并在github上添加公钥即可。

输入git push -u origin master完成本地代码上传，结果见下图

从github服务器拉取最新的代码可以使用命令git pull origin master

总结

echo "# xx" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin git@github.com:仓库地址.git
git push -u origin master
